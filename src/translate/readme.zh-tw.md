# 翻譯

## 設置

翻譯服務由[百度翻譯開放平臺](https://fanyi-api.baidu.com/)提供，請註冊好您的開發賬號後，填寫以下開發者信息方可使用。

:[appid(APP ID) secret(密鑰)]

您的 APP ID 和密鑰只保存在您瀏覽器的 LocalStorage 中，直至您清空瀏覽器緩存會一直保存，不用重複設置。**我們不會儲存您的任何賬號信息。**

如果您的密鑰有重置，請記得要重新設置密鑰信息，以免插件無法使用。

### 如何找到開發者信息

登陸您的[百度翻譯開放平臺](https://fanyi-api.baidu.com/)賬號，點擊“管理控制檯”，然後在左邊選擇“開發者信息”，就可以看到需要的 APP ID 和密鑰了。
